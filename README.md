# Bootstrap Detector

Instantly detect Bootstrap on any webpage.

## Description

Bootstrap Detector is a browser extension that quickly identifies whether a website is using the Bootstrap framework. With a sleek, modern interface, it provides instant results with a simple click.

## Features

- One-click Bootstrap detection
- Elegant, animated user interface
- Clear visual indicators for detection results
- Lightweight and fast

## Installation

1. Clone this repository or download the ZIP file.
2. Open your browser's extension management page:
   - Chrome: `chrome://extensions`
   - Firefox: `about:addons`
   - Edge: `edge://extensions`
3. Enable "Developer mode".
4. Click "Load unpacked" and select the extension directory.

## Usage

1. Navigate to any website you want to check.
2. Click the Bootstrap Detector icon in your browser toolbar.
3. Wait for the detection result.

## Files

- `manifest.json`: Extension configuration
- `background.js`: Background script for Bootstrap detection
- `popup.html`: HTML structure for the popup
- `popup.css`: Styles for the popup interface
- `popup.js`: JavaScript for popup functionality
- `icon.png`: Extension icon (16px, 48px, and 128px sizes)

## Development

To modify the extension:

1. Edit the relevant files.
2. Reload the extension in your browser's extension management page.
3. Test the changes by clicking the extension icon.
